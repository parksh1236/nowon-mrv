# 노원구 MRV 측정·검증·계산

노원구 에너지정보플랫폼(nowon-energy-platform)의 탄소중립계획 중 **MRV 부분**을 분리한 독립 프로젝트입니다.

## 구성

- `index.html` — 단일 페이지 앱
  - MRV(측정·보고·검증) 개념 및 3단계 설명
  - **감축량 계산기**: 가이드라인 8개 부문 135개 감축사업 원단위 기반, 부문 → 사업 → 산정기준 선택 + 다중 항목 합산
  - 노원구 계획 주요 감축원단위 표 (10개)
- `mrv.json` — 데이터 파일 (원본 `platform/carbon.json`에서 `mrv` 객체를 추출)

## 실행

정적 파일 서버로 열면 됩니다.

```bash
npx http-server . -p 8737
# http://localhost:8737
```
