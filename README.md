# 탄소중립 기본계획 플랫폼 — 비교 · 평가 · MRV

지자체 탄소중립 녹색성장 기본계획의 **검색·비교**, **프레임워크 평가**, **MRV 감축량 계산**을 제공하는 단일 페이지 웹앱입니다. (노원구 MRV 프로젝트에서 확장)

## 구성 (3탭)

- `index.html` — 단일 페이지 앱
  - **탭1 검색·비교**: 지자체 기본계획 핵심 지표(목표·배출량·부문·예산·평가등급)를 최대 3개 나란히 비교
  - **탭2 프레임워크 평가**: 통합 평가 프레임워크(10대 기준 × 10점) 기준 지자체별 평가 결과 + 근거 + 반영자료·이력. 추가 자료를 드라이브에 넣고 재평가를 요청하면 평가가 갱신됨(반자동)
  - **탭3 MRV 계산기**: 가이드라인 8개 부문 135개 감축사업 원단위 기반 감축량 계산
- `mrv.json` — MRV 계산기 데이터
- `data/framework.json` — 통합 평가 프레임워크 v3.4 (10대 기준 루브릭)
- `data/plans.json` — 지자체 기본계획 추출 데이터 (현재 5개: 노원구·강남구·강북구·성남시·동두천시)
- `data/evaluations.json` — 프레임워크 평가 결과 (점수·근거·반영자료·이력)

## 실행

정적 파일 서버로 열면 됩니다.

```bash
npx http-server . -p 8737
# http://localhost:8737
```
